# Math Coursework Calculator

A Python application that calculates:

- Midterm grade
- Assignment grade
- Quiz grade
- Overall coursework grade

## Features
- Input validation
- Automatic grade scaling
- User-friendly terminal interface
- Executable (.exe) version available

## Technologies Used
- Python 3
